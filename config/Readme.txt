Here are some ADTF projects contained which can be used as templates or examples

Files:
* start_PythonConfig.sh: 					starts the Project for the example with tensorFlow integration for linux
* start_UserProject.sh: 					starts the Project for the User Project for linux
* start_CameraCalibrationConfig.sh: 			starts the Project for camera calibrations for linux
* start_LiveVisualizationConfig.sh: 			starts the Project for the LiveVisualization for linux
* start_RecordPlaybackConfig.sh: 				starts the Project for the RecordandPlayback for linux

* start_PythonConfig.bat: 				starts the Project for the example with tensorFlow integration for windows
* start_UserProject.bat: 					starts the Project for the User Project for windows
* start_CameraCalibrationConfig.bat: 			starts the Project for camera calibrations for windows
* start_LiveVisualizationConfig.bat: 			starts the Project for the LiveVisualization for windows
* start_RecordPlaybackConfig.bat: 				starts the Project for the RecordandPlayback for windows
